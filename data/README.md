# Data
Ce dossier contient tous les fichiers nécessaire pour le traitement via DESq2 :
- Aureo_data comporte la correspondance gène et nom pour le MA plot des gènes liés à la traduction (les gènes liés à la traduction sont obtenus via KEGG tandis que les noms sont obtenus par une correspondance avec un fichier AureoWiki: celui ci-dessous)
- GeneSpecificInformation est le fichier contenant les noms de tous les gènes (obtenu via AureoWiki)
- GSE139659 est le fichier contenant les résultats des expériences des chercheurs de l'article

Ce dossier comprend aussi les scripts R nécessaire à la réalisation des figures.
